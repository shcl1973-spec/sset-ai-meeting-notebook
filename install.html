同步架構說明

同 Wi‑Fi 使用時，手機會連到這台電腦的區網網址。若把此專案部署到雲端伺服器，同一個同步 API 可讓手機與電腦共用資料。

注意：手機瀏覽器真正安裝 PWA 通常需要 HTTPS。區網網址適合測試與內部使用；正式給業務安裝建議部署到 HTTPS 雲端。

  1. 手機掃 QR Code 開啟 App。
  2. 在 App 的「同步」面板填入同步 API。
  3. 按「智慧同步」即可合併手機與電腦資料。
