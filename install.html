手機無法使用時

  1. 手機必須和電腦在同一個 Wi-Fi。
  2. 手機不能使用 127.0.0.1，請掃描區網網址，例如 http://192.168.x.x:8000/。
  3. 如果區網網址也打不開，請檢查 Windows 防火牆是否允許 Node.js 或連接埠 8000。
  4. 手機錄音與 PWA 正式安裝通常需要 HTTPS；區網 HTTP 適合測試，正式使用建議部署到 HTTPS 雲端。

直接安裝到手機

  1. 正式安裝請使用 HTTPS 網址，例如 GitHub Pages。
  2. iPhone：用 Safari 開啟網址，按分享，選「加入主畫面」。
  3. Android：用 Chrome 開啟網址，選「安裝應用程式」或「加入主畫面」。
  4. 安裝後第一次開啟，在「雲端同步」貼上 shcl1973 Google Apps Script 的 /exec 同步網址。

同步架構說明

同 Wi-Fi 使用時，手機會連到這台電腦的區網網址。若使用 shcl1973 Google 雲端硬碟的 Apps Script 同步端點，手機與電腦可跨網路共用資料。

  1. 手機掃 QR Code 開啟 App。
  2. 在 App 的「同步」面板填入同步 API。
  3. 按「智慧同步」即可合併手機與電腦資料。
